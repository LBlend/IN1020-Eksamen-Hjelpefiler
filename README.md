# Hjelpefiler til IN1020 eksamen

Noen filer som løser evt. eksamensoppgaver til eksamen Høst 2020. Fordelen med hjemmeeksamen er at man ikke trenger å bruke hodet :)

Felles for alle disse er at de ikke dekker edge cases og vil dermed gi ut feil svar i noen tilfeller. Skal ikke stoles blindt på. Koden er heller ikke skrevet for å være ryddig og pen.
